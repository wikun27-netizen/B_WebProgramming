Cara upload ke github



  1. Di terminal -->"cd(change directori) -nama_file-" Pindah file.
  2. -->"git init" Inisialisasi local repository.
  3. -->"remote add origin -https_address_online_repository.git" set upload location.
  4. -->"git status" Lihat status file.
    • -->"git add -file_name-" save 1 file.
    • -->"git add *" save semua file.
  6. -->"git commit -m '-pesan-'" commit ke local repository.
  7. -->"git push -u origin master" commit ke github.


Note :

    -->"touch -file_name-" buat bikin file baru.

    -->"pip install -package_name-==-version- install package dengan versi tertentu

    -->"pip list" lihat package yang sudah diinstall

    -->"pip install --default-timeout=100 -package_name- install package dengan waktu timeout lebih lama
